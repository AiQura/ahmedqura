Auto-mpg dataset

The first step in learning machine learning is to perform EDA for the data you have and preprocess it, during the unit 2 seminar, the main concentration was to know how to preprocess the data so it will be ready for the model to be trained to, this preprocessing included investigating the data the Skewness, the Heat Map, and correlations, a tutorial for house prices was given along with EDA for it, then we had to perform the same analysis to Auto-mpg dataset, the EDA aims are many among them are identifying the missing values and replace categorical data with numerical data.
